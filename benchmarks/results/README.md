# Benchmark Results

Curated benchmark summaries live here so jumpyBrain retrieval changes can be compared over time.

## Policy

- Store summaries, metrics, commands, and high-level notes.
- Do not store raw external benchmark data here.
- Do not store raw retrieval JSONL with dataset snippets here unless explicitly approved.
- Raw/local outputs should remain under ignored `bench-results/`.
- Downloaded datasets should remain under ignored `benchdata/`.

## Layout

```text
benchmarks/results/
  longmemeval/
    YYYY-MM-DD-<run-name>.md
```
